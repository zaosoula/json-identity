# Tattoo

```json
{
  "bodyLocation": null,

}
```
---
Model inspired by [REV23](https://www.rev23.com/products/software/rev23-desktop/)

### Properties

- `bodyLocation` (string)
- `designDescription` (string)
- `width` (float) - in centimeters
- `height` (float) - in centimeters
- `colors` (array(string))
- `usage` ([usage](../usage))
